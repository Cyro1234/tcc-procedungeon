<h1 align="center">Alex: Road to Home</h1>

> **Status do Projeto:** 🚀 Em Desenvolvimento

Este repositório contém o código-fonte e os sistemas desenvolvidos para o Trabalho de Conclusão de Curso (TCC) em **Ciência da Computação** do **Centro Universitário do Instituto Mauá de Tecnologia (IMT)**[cite: 1, 2, 7].
O objetivo principal deste projeto é pesquisar, implementar e avaliar o impacto da Geração Procedural de Conteúdo de forma **totalmente integrada** nos três pilares de um jogo *Roguelike*: **mapas, inimigos e itens** [cite: 11, 14, 31][cite_start], distanciando-se das abordagens comerciais comuns que limitam o aspecto procedural apenas ao layout das fases[cite: 14, 30].
